gitbook的安装及使用

安装

  npm install -g gitbook-cli

查看版本

  gitbook -V

使用

  gitbook init
  使用gitbook init初始化gitbook,会生成两个文件
      - README.md ---- 书籍的介绍
      - SUMMARY.md ---- 数据的目录结构
  再次使用gitbook init,会通过SUMMARY.md文件描述生成具体的目录和文件
  gitbook serve
  使用gitbook serve来预览本书籍,默认为4000端口,也可利用--port XXXX来设置具体端口

存在问题

  因为node版本问题进而在初始化时出现
  Error loading version latest: Error: Cannot find module 'internal/util/types'错误,
  解决办法:
      对node进行降级,或者切换低版本的node

gitbook其他相关命令

  - build指定路径
      gitbook build [书籍路径] [输出路径]
  - serve指定端口
      gitbook serve --port 2333
  - 生成PDF格式的电子书
      gitbook pdf ./ ./mybook.pdf
  - 生成epub格式的电子书
      gitbook epub ./ ./mybook.epub
  - 生成mobi格式的电子书
      gitbook mobi ./ ./mybook.mobi

gitbook插件

  - -search                                   去除默认自带的插件
  - back-to-top-button                        回到顶部
  - expandable-chapters-small                 可扩展导航章节
  - insert-logo                               插入logo
  - chapter-fold                              导航目录折叠
