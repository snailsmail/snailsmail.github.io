weex的安装及使用

Weex 是使用流行的 Web 开发体验来开发高性能原生应用的框架。

首先安装工具 (安装最新的beta版本工具,weex-toolkit只支vue.js项目)

    npm install -g weex-toolkit@beta 
    sudo ln -s xxx/node/bin/weex /usr/local/bin/weex

查看当前weex工具版本

    weex -v

初始化项目

    weex create awesome-project
    执行后生成awesome-project目录的标准项目结构

开发

    进入路径,先安装依赖 npm install,之后直接运行 npm start 就可以将项目完整的跑起来

编译和运行

    通过 weex platform add 来添加特定平台项目
        weex platform add ios
        weex platform add android
    想在本地机器上打开android或ios项目,需要配置客户端的开发环境.对于ios,本地需要安装Xcode
    对应android,本地需要安装android studio.
    当开发环境准备就绪后,运行下面的命令.可在模拟器或真实设备上启动应用
        weex run ios
        weex run android
        weex run web

调试

    weex debug 这条命令会启动age调试服务,并只支持V8引擎的桌面浏览器,可在chrome中打开调试页面
